# RsaPractice
# Журнал тестирования
| № | Проверка | Действие | Ожидание | Фактический результат | Статус |
|---|----------|----------|----------|----------------------|--------|
| 1 | Валидация ввода простых чисел | В поле "Мин. простое" ввести букву `a` | Символ не появляется, ввод блокируется | Ввод букв заблокирован, отображаются только цифры | passed |
| 2 | Некорректный диапазон простых чисел | Мин = 5, Макс = 3, нажать «Сгенерировать ключи» | Сообщение: «Минимальное значение должно быть >= 2 и меньше максимального» | Выводится сообщение об ошибке с указанным текстом | passed |
| 3 | Шифрование без сгенерированных ключей | Не нажимая «Сгенерировать ключи», сразу нажать «Зашифровать» | Появляется окно: «Сначала сгенерируйте ключи.» | Появляется информационное окно с соответствующим сообщением | passed |
| 4 | Шифрование пустой строки | Сгенерировать ключи, оставить поле «Открытый текст» пустым, нажать «Зашифровать» | Поле «Шифротекст» остаётся пустым, ошибок нет | Шифротекст пуст, исключений не возникает | passed |
| 5 | Шифрование символа с кодом больше n | Сгенерировать ключи с мин=2, макс=3 (n=6), ввести символ «A» (код 65) и нажать «Зашифровать» | Сообщение об ошибке: «Код символа ... больше или равен модулю» | Выводится MessageBox с текстом ошибки о превышении модуля | passed |
| 6 | Дешифрование пустого шифротекста | Сгенерировать ключи, оставить поле «Шифротекст» пустым, нажать «Дешифровать» | Сообщение: «Нет данных для дешифрования. Сначала зашифруйте текст.» | Появляется предупреждающее окно с указанным текстом | passed |
| 7 | Дешифрование некорректных данных | В поле «Шифротекст» (только чтение) вставить строку «123,abc,456» и нажать «Дешифровать» | Сообщение об ошибке формата: «Блок 'abc' не является целым числом» | Выводится сообщение об ошибке формата с указанием проблемного блока | passed |
| 8 | Полный цикл с русским текстом | Сгенерировать ключи (100-200), ввести «Привет, мир!», зашифровать, затем дешифровать | После дешифрования в поле «Расшифрованный текст» появляется «Привет, мир!» | Расшифрованный текст полностью совпадает с исходным | passed |
