## 1. Описание предметной области
Асимметричный алгоритм шифрования RSA в учебной реализации. Шифрование выполняется посимвольно над байтами UTF-8 строки. Каждый байт возводится в степень открытой экспоненты `e` по модулю `n`. Дешифрование возводит шифроблок в степень закрытой экспоненты `d` по тому же модулю. Ключи генерируются на основе двух случайных простых чисел из заданного диапазона. Простота алгоритма позволяет продемонстрировать принцип работы RSA.
